## 第三步：下载 SSH 密钥（这步最关键）

这是 DMIT 如何登录服务器这件事里最容易被忽视的一步，也是新手踩坑率最高的地方。

**DMIT 默认不支持密码登录 VPS，必须用 SSH 密钥文件。**

VPS 开通后（购买完成几分钟内会自动开通），按照以下步骤操作：

1. 登录 DMIT 客户中心
2. 点击顶部导航的 **Services**（服务）
3. 找到你购买的 VPS，点击进入详情页
4. 首次进入时会弹出下载密钥的提示；或者点击页面右上角的 **SSH 金钥管理**
5. 点击 **Download SSH Key**（下载 SSH 密钥）按钮，下载压缩包

**非常重要：SSH 密钥只能下载一次。** 没有第二次机会。下载后请立刻备份到安全的地方，比如网盘。如果丢失了，只能在后台重新生成，旧密钥立即失效，相当于换了一把锁。

解压下载的压缩包，你会看到两个文件夹：`Private_key` 和 `Public_key`。你需要的是 `Private_key` 文件夹里的这两个文件：
- `id_rsa.pem`：用于 XShell、MobaXterm、Mac/Linux 终端
- `id_rsa.ppk`：用于 PuTTY（Windows 常用工具）

## 第四步：用 SSH 密钥登录 VPS 服务器

拿到密钥文件之后，根据你的系统和工具选择对应的方式。

### Windows 用户（推荐 XShell 或 MobaXterm）

**用 XShell：**

1. 打开 XShell，点击"工具"→"用户密钥管理者"
2. 点击"导入"，选中 `id_rsa.pem` 文件，成功导入
3. 新建会话，填入 VPS 的 IP 地址，端口默认 22，用户名填 `root`
4. 在身份验证方式选择"Public Key"（公钥），选择刚才导入的密钥
5. 点击连接，成功进入服务器

**用 PuTTY：**

1. 打开 PuTTY，在"Host Name"处填入 VPS 的 IP 地址
2. 点击左侧"Connection"→"Data"，在"Auto-login username"填 `root`
3. 点击"Connection"→"SSH"→"Auth"，在"Private key file"处选择 `id_rsa.ppk` 文件
4. 返回"Session"，保存会话，点击"Open"连接

### Mac / Linux 用户（直接用终端）

打开系统自带终端，运行以下命令：

bash
chmod 600 /path/to/id_rsa.pem
ssh -i /path/to/id_rsa.pem root@你的VPS_IP


第一行是给密钥文件设置正确权限——SSH 会拒绝权限太宽松的密钥文件，这步执行一次就够了。第二行是连接命令，把路径和 IP 换成实际的就行。

### 手机用户（Android）

使用 JuiceSSH 这款 App，将 `id_rsa.pem` 文件从电脑传到手机，在 JuiceSSH 的密钥管理里导入，然后创建连接即可。

## 如何开启密码登录（可选，不推荐新手折腾）

DMIT 禁用密码登录是出于服务器安全考虑。但如果你实在习惯用密码登录，也可以在成功用密钥登录后，手动修改服务器的 SSH 配置来开启。

这需要你已经能成功用密钥连进服务器。连进去之后编辑 `/etc/ssh/sshd_config` 文件，找到 `PasswordAuthentication no`，改成 `PasswordAuthentication yes`，然后重启 SSH 服务。具体步骤因系统不同略有差异，Debian/Ubuntu 和 CentOS 操作大同小异。

**如果你决定开启密码登录，请务必设置一个强密码**（建议12位以上，混合大小写字母、数字和特殊符号）。弱密码在公网上基本等于没有密码。

## 常见问题：DMIT 登录卡住了怎么办

### Q：登录客户中心总是显示密码错误？

先确认邮箱输入正确。然后用"忘记密码"功能重置一次。如果重置邮件没收到，检查垃圾邮件文件夹。

### Q：SSH 连接时提示"Permission denied (publickey)"？

大概率是以下几种情况之一：密钥文件没有设置正确权限（Mac/Linux 用户需要 `chmod 600`）；选错了密钥文件（PuTTY 用 .ppk，其他工具用 .pem）；IP 地址填错了。逐一检查。

### Q：密钥文件丢失了怎么办？

登录 DMIT 客户中心，进入对应 VPS 的详情页，在 SSH 金钥管理里重新生成密钥，然后重新下载。注意：重新生成后旧密钥立即失效，必须用新密钥重新连接。

### Q：VPS 的 IP 在哪里找？

两个地方都可以找到：一是购买完成后 DMIT 发到你邮箱的开通邮件里；二是登录客户中心，在对应 VPS 的服务详情页里。

### Q：支持中文客服吗？

支持。DMIT 有中国人参与运营，提交工单时可以用中文写，对方通常以英文回复。
